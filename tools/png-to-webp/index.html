User Manual

  1. Drop one or more PNG files onto the upload area.
  2. Adjust the WebP quality slider (85% is a great default).
  3. Click Convert All to WebP.
  4. Each file downloads automatically with a .webp extension.

Architecture Insight

Each PNG is decoded using a FileReader, drawn onto a hidden Canvas, then re-encoded via canvas.toBlob('image/webp', quality). The browser's native WebP encoder handles the conversion — no libraries, no WASM, no server round-trips.

FAQ

Does converting PNG to WebP lose transparency?
WebP supports full alpha transparency, so transparent PNGs are preserved. The quality slider controls lossy compression of the colour channels.
How much smaller will WebP be vs PNG?
Typically 25–35% smaller for photographic images. For diagrams or screenshots with flat colours, savings may be lower.
Is there a file size limit?
No hard limit — it depends on your browser's available memory. Files up to ~50 MB convert without issues on modern browsers.
