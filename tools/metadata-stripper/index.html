📖 User Manual

  1. Drop one or more JPEG or PNG files onto the zone.
  2. Click Strip Metadata & Download.
  3. Each file is processed and downloads automatically — fully stripped of all EXIF data.

Metadata stripped includes: camera model, lens, ISO, GPS location, creation date, author, copyright, and all other EXIF/IPTC/XMP tags.

⚙️ Architecture Insight

The stripping method re-draws each image onto a blank Canvas element, then exports the canvas back to JPEG or PNG using toBlob(). The browser codec writes only pixel data to the new file — all metadata containers (EXIF, IPTC, XMP) are discarded. No data ever leaves your browser.

❓ FAQ

Does this remove GPS location from my photos?
Yes. The Canvas re-export completely discards all EXIF data including GPS coordinates, altitude, and bearing. See also the dedicated GPS Location Remover tool.
Will my image quality change?
For JPEG output a quality of 95% is used, which is visually lossless for most photos. PNG output is lossless.
Does this tool upload my images?
Absolutely not. Everything runs inside your browser. No data is sent to any server at any point.
Can I use this to prepare images before posting on social media?
Yes — this is one of the most common use cases. Strip location and device data before uploading to any platform.
