User Manual

  1. Drop photos (JPEG or PNG) onto the upload zone.
  2. Click Remove Location & Download.
  3. Each photo re-downloads with all GPS/location data removed.

This tool removes all EXIF data, not just GPS. Use the Metadata Stripper if you want a more explicit bulk-EXIF tool.

Architecture Insight

Location data in photos is stored as GPS EXIF tags embedded in the JPEG/PNG container. This tool re-renders the image pixel-by-pixel onto a Canvas and exports using toBlob(). The new file contains only raw pixel data — no EXIF container, no GPS, no metadata of any kind.

FAQ

Will my photo look the same after processing?
Yes. For JPEG, a quality of 95% is used which is visually identical to the original. PNG output is fully lossless.
Does this work for photos taken on iPhone/Android?
Yes. Modern smartphones embed GPS in JPEG EXIF headers. This tool removes those tags completely.
Is this safe for sensitive photos?
Completely. Everything runs in your browser's sandbox. No data is transmitted, logged, or stored anywhere outside your device.
Does removing GPS location affect Google Photos or iCloud?
If you upload the stripped photo, those services won't be able to read GPS from the EXIF. Their map/places features rely on the embedded coordinates.
